# Release 0.7.0

**Type:** minor (new user-facing feature since 0.6.0).

## Headline

- **Per-session transcript chat ("Ask about this transcript")** — ask the active
  AI provider (OpenAI/Gemini) about a saved transcript; history persisted per
  session.
- **Collapsible "Ask" sidebar** — chat shown as a third shell column on the
  session-detail page (history · transcript/summary · chat); collapses to a thin
  vertical "Ask" strip, choice persisted in `configStore` (`chatCollapsed`).

## Also since 0.6.0 (internal)

- TDD foundation (Rust `cargo test` + frontend `bun test` suites).
- CI quality gates: tests + ESLint enforced; added `dependency-audit` +
  `commit-lint` jobs.
- Decomposition: split `commands.rs` / `db.rs` / `ai.rs` into per-domain modules;
  extracted session-detail and settings helpers.
- Windows updater prefers NSIS; `CLAUDE.md` untracked.
