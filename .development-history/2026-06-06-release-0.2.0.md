# Release 0.2.0

**Goal:** Cut the second release after 0.1.0, shipping the features accumulated
since the first public release.

## What shipped (since v0.1.0)

- **Added:** About page; configurable AI summary output language; rename speaker
  labels from a session's history; adjustable background transparency.
- **Changed:** new brand identity — bullseye app icon (window/taskbar/installer)
  - wordmark logo in the titlebar; README/favicon updated.

## Process

- Bumped version → `0.2.0` in `package.json`, `src-tauri/tauri.conf.json`,
  `src-tauri/Cargo.toml`, and `src-tauri/Cargo.lock` (synced via `cargo check`).
- Updated `CHANGELOG.md` ([0.2.0] section + compare links).
- Verified locally: `cargo check` ✓ and `bun run build` ✓.
- Commit `chore(release): 0.2.0`, annotated tag `v0.2.0` → the Release workflow
  (`tauri-action`) built the Windows installers on `windows-latest`.

## Result

- Draft GitHub Release `v0.2.0` created with assets:
  - `sososo_0.2.0_x64-setup.exe` (NSIS, ~4.9 MB)
  - `sososo_0.2.0_x64_en-US.msi` (~6.6 MB)
- Pending a manual **Publish**.

## Notes

- Releases are draft-by-default (`release.yml` → `releaseDraft: true`); publish
  via the Releases page or `gh release edit v0.2.0 --draft=false`.
- Builds are not code-signed yet → Windows SmartScreen may warn on first run.
