# Release 0.5.0

**Goal:** Ship the features added since v0.4.0.

## What shipped (since v0.4.0)

- **Added:** rich Markdown rendering in AI session summaries; a summary-language
  selector + transcript translation in the session-detail view.
- **Fixed:** contact email across the Code of Conduct, Privacy, and Security docs.
- **Docs:** README gained a prominent per-OS Download section; demo placement
  reworked (not part of the build).

## Process

- Bumped `0.4.0` → `0.5.0` in `package.json`, `src-tauri/tauri.conf.json`,
  `src-tauri/Cargo.toml`, `src-tauri/Cargo.lock` (synced via `cargo check`).
- Updated `CHANGELOG.md` ([0.5.0] section + compare links).
- Verified locally: `cargo check` ✓ and `bun run build` ✓ (81 modules).
- Commit `chore(release): 0.5.0`, annotated tag `v0.5.0` → the Release workflow
  builds **Windows + macOS (universal) + Linux** and attaches all bundles.

## Result

- Tag `v0.5.0` pushed → multi-OS Release workflow produces a **draft** GitHub
  Release (`releaseDraft: true`) for manual Publish after review.

## Notes

- Multi-agent repo: only the 5 release files were staged (explicit paths); no
  other agent's work was touched.
